Context

Problem & Solution

⚠ The Problem

Manual pathologist review of H&E stained histology slides is time-consuming, expensive, and subject to significant inter-observer variability. High-throughput screening pipelines are blocked by this bottleneck.

✓ The Solution

A fine-tuned ResNet-50 classifier that processes a slide image in under 2 seconds and outputs a class probability with a Grad-CAM heatmap overlay, enabling rapid first-pass screening and transparent decision support.

Methodology

Technical Approach

01

Data Collection & Preprocessing

Downloaded 2,400 labeled H&E slide patches from the TCGA dataset (1,200 tumor, 1,200 normal). Applied stain normalization using Macenko's method and tiled each WSI into 256×256px patches at 20× magnification.

02

Model Architecture & Training

Used ImageNet-pretrained ResNet-50 with a custom classification head. Trained with AdamW optimizer (lr=1e-4), cosine annealing scheduler, and aggressive augmentation (random flip, rotation, color jitter, elastic deformation) over 30 epochs.

03

Explainability with Grad-CAM

Implemented Grad-CAM++ on the final convolutional block to generate per-prediction heatmaps. Heatmaps are upsampled to original image size and blended at 40% opacity. Validated that model attends to clinically relevant regions (nuclear atypia, mitotic figures) confirmed by a pathologist collaborator.

04

Deployment

Packaged as a Streamlit app with drag-and-drop image upload, class probability bar chart, and downloadable Grad-CAM overlay. Containerized with Docker for reproducibility. Also exposes a FastAPI endpoint for programmatic access.

import torch
from torchvision import transforms
from pytorch_grad_cam import GradCAMPlusPlus

def predict_with_gradcam(model, image_path, device="cuda"):
    """Run inference and return class + Grad-CAM heatmap."""
    transform = transforms.Compose([
        transforms.Resize((256, 256)),
        transforms.ToTensor(),
        transforms.Normalize(mean=[0.485, 0.456, 0.406],
                             std=[0.229, 0.224, 0.225])
    ])
    img   = transform(Image.open(image_path)).unsqueeze(0).to(device)
    model = model.to(device).eval()

    cam   = GradCAMPlusPlus(model=model,
                              target_layers=[model.layer4[-1]])
    mask  = cam(input_tensor=img)[0]   # (H, W) heatmap

    with torch.no_grad():
        logits = model(img)
        probs  = torch.softmax(logits, dim=1).cpu().numpy()[0]

    return {"class": CLASSES[probs.argmax()],
            "confidence": float(probs.max()),
            "heatmap": mask}

Learnings

Challenges & How I Solved Them

🎨

Stain Variability Across Institutions

H&E staining protocols differ between labs, causing distribution shift. Solved by applying Macenko stain normalization at the preprocessing step, reducing cross-site accuracy drop from 12% to 3%.

⚖️

Class Imbalance in Edge Cases

Borderline lesion patches were underrepresented. Addressed with focal loss (γ=2) and oversampling of the minority class during training, improving recall from 81% to 91% on difficult cases.

⚡

Inference Speed for Whole-Slide Images

WSIs contain thousands of patches; naive inference was too slow. Implemented batched tile processing with a CUDA-enabled DataLoader, reducing whole-slide inference from 4 minutes to 22 seconds.
